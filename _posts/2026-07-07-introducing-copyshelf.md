> **TL;DR**: 传统的剪贴板工具主要解决“历史记录”和“快速取回”的问题，但对“内容格式理解”、“目标应用适配”和“多语言排版兼容”的支持较弱。**CopyShelf** 是一款专为 macOS 打造的智能剪贴板管理工具。它不仅能保存和调用复制历史，还能理解内容格式、判断目标输入场景，并以最合适的形式完成粘贴与转换。

在日常开发、内容创作与跨应用协作中，我们频繁地在不同的应用之间进行“复制、切换、再粘贴”的操作。然而，我们经常遇到以下尴尬的情况：
1. **Markdown 与富文本的格式丢失**：从 Markdown 编辑器复制一段排版，粘贴到 Apple Notes（备忘录）时只剩下一堆无意义的原始符号或纯文本；或者反过来，从 Apple Notes 复制的格式粘贴到 `.md` 文件时乱成一团。
2. **多语言排版混乱**：在处理阿拉伯语、希伯来语等 RTL（Right-to-Left，从右至左）文本时，一旦混排了英文、数字或标点，方向便会错乱、标点符号错位。
3. **机械的手动格式转换**：每次粘贴都需要人手动介入清洗格式，降低了整体的工作效率。

**CopyShelf** 的诞生正是为了解决这些高频却一直被忽视的剪贴板痛点。

## 🛠️ CopyShelf 的核心功能与差异化

### 1. Markdown 与 Apple Notes 智能双向转换
在 Markdown 与 Apple Notes 富文本之间提供高频、低心智负担的双向转换体验。
* 当您在 Markdown 环境（如 VS Code、Obsidian）复制内容并切换到 Apple Notes 粘贴时，CopyShelf 会智能输出 Notes 可识别的标题、列表、粗体与链接排版。
* 当您从 Apple Notes 复制富文本内容粘贴到 `.md` 编辑器时，CopyShelf 会自动将其转换并输出为规范的 Markdown 语法。

### 2. 目标应用与场景感知 (App & Context Aware)
CopyShelf 会自动判断您当前处于哪种输入场景：
* 在 Markdown 编辑场景下，它自动优先输出 **Markdown 文本**。
* 在富文本编辑场景下，它自动优先输出 **富文本样式**。
* 您完全不需要每次都手动切换格式，系统会“自动理解并一键确认”。对于特殊情况，您也可以通过快捷键手动指定粘贴模式。

### 3. RTL 文本方向修正与混排优化
面向多语言用户，CopyShelf 提供了阿拉伯语、希伯来语等 RTL 文本的方向修正与顺序适配。彻底解决中英与中东语言混排时，标点错位、数字混乱的显示顽疾。

### 4. 毫不妥协的剪贴板基础能力
CopyShelf 完美对齐并优化了主流 macOS 剪贴板工具（如 Pastebot）的核心体验，确保日常使用的绝对稳定：
* **最近历史与搜索**：轻量快捷，支持按文本、富文本、链接、图片等多种数据类型检索。
* **常用条目固定 (Pinned Lists)**：支持分类管理您的常用文本片段或常用集合。
* **Quick Paste Menu**：通过全局快捷键迅速唤起浮窗，回车或双击即可实现粘贴。
* **隐私黑名单**：智能忽略来自 1Password、Keychain 等密码管理器的复制历史，保护敏感数据安全。
